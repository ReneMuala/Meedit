* Follow this little tutorial to build Meedit in your machine.

NOTE: do that after install GCC/G++, cmake and ncurses.

#1 Download Meedit

#2 Open terminal and make sure that you are in Meedit folder

#3 type make [enter]

Congratulations!
Now, you have sucessful created Meedit 0.2 in your machine 
Have fun!

you can run it by typing "./Meedit" [enter]
